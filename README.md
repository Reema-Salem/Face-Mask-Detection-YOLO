# Face Mask Detection using YOLO

## Overview
This project develops a deep learning system for detecting whether people are wearing face masks using YOLO. The model performs real-time object detection and classifies faces as Mask or No Mask.

## Status
🚧 Work in Progress

## Key Features
- Real-time face mask detection
- Detects multiple faces in one image
- Classifies each face (Mask / No Mask)
- High speed and efficiency using YOLO

## Methodology
- Preprocessing: resizing, normalization, augmentation  
- Model: YOLO (object detection)  
- Training: labeled dataset with bounding boxes  
- Evaluation: Precision, Recall, mAP  

## Technologies
- Python  
- YOLO  
- OpenCV  
- TensorFlow / PyTorch  

## Results
- Training in progress  
- Expected strong real-time performance
